sims4mcp

CLI and MCP adapter for inspecting save files

sims4mcp parses binary save files and exposes the data through a command-line interface and an MCP server. Extract Sims, households, traits, careers, and relationships without opening the game.

Install

pip install -e .

CLI usage

CommandWhat it does
sims4mcp info <save>Overview — households and sim counts
sims4mcp sims <save>Detailed per-sim data
sims4mcp sims --sim-id N <save>Single sim by ID
sims4mcp households <save>Household list with members and funds

If no path is given, the latest save from the default directory is used.

# Auto-detect latest save
sims4mcp info

# Inspect a specific file
sims4mcp sims ~/saves/slot_00000002.save

# Filter by sim ID
sims4mcp sims --sim-id 123456 ~/saves/slot_00000002.save

MCP server

Start the server over stdio:

mcp run src/sims4mcp/mcp_server.py

Exposed tools:

  • list_households — all households with members and funds
  • list_sims — all sims, optional household filter
  • get_sim — full details for one sim (by ID or name)
  • load_save — switch to a different save file

API

from sims4mcp import load_save, SaveFile, Sim, Household

sf = load_save("path/to/save.save")
for hh in sf.households:
    print(hh.name, hh.funds)
    for sim in hh.sims:
        print(f"  {sim.first_name} — {sim.age}")

Data model

EntityFields
SaveFilepath, name, households, sims (flattened)
Householdid, name, funds, sims
Simid, name, age, gender, traits, career, skills, moodlets, money, relationships

Limitations

  • POC — binary field offsets may need tuning against real save files
  • Multi-sim saves: trait/career data uses the first resource found rather than filtering by sim instance
  • Read-only — no write or patch support
  • Game updates can shift the format
